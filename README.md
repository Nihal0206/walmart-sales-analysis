# Walmart Sales Data Analysis (Python & SQL)

## Overview
This project analyzes Walmart sales data using Python and SQL to identify sales trends and basic business insights. The project was created for learning and practice in data analysis and SQL querying.

## What I Did
- Explored and cleaned Walmart sales data using Python
- Performed analysis using SQL queries (aggregation and grouping)
- Analyzed sales trends by category, branch, and payment method

## Tools Used
- Python (Pandas)
- SQL (MySQL / PostgreSQL)
- Kaggle Dataset

## Learning
- Data cleaning and preparation
- Writing SQL queries for business analysis
- Using Python and SQL together for data analysis
